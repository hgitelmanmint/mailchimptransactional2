<Update label="1.3" description="Released January 2026">
  ## Version 1.3

  - Added `spam` and `reject` webhook event types.
  - `/messages/info` now returns per-message `clicks` and `opens` counts inline.
  - Improved `reject_reason` detail on invalid recipients.
</Update>

<Update label="1.1" description="Released October 2025">
  ## Version 1.1

  - Introduced structured per-recipient send results with stable `_id` values.
  - Added `track_clicks` granularity at the message level.
  - First minor version release since the product re-entered active development.
</Update>

<Update label="1.0" description="Legacy">
  ## Version 1.0

  - Original Mandrill-era API. Still supported. See [Migrating from 1.0](/guides/migrating-from-1-0).
</Update>
